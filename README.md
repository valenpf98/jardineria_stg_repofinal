# Proyecto Jardinería - Staging

Este repositorio reúne todas las actividades desarrolladas en el curso **Bases de Datos II**, incluyendo la creación de la base de datos *Staging*, los scripts SQL, informes y pruebas de calidad de datos.

## Contenido del repositorio

- **informes/**
  - Documentos en PDF de las entregas anteriores (EA1 y EA2).

- **scripts_sql/**
  - `CreacionJardineria.sql`: Script de creación de la base original.
  - `CreacionBDJardineria_staging.sql`: Script de creación de la base *Staging*.
  - `CargaBDJardineria_staging.sql`: Script de carga de datos en *Staging*.
  - `ValidarCargaStaging.sql`: Consultas de validación de la carga.

- **pruebas_calidad_datos/**
  - `prueba_calidad_datos.sql`: Consultas para verificar calidad de datos (nulos, duplicados, rangos inválidos, referencias huérfanas).

## Objetivo

El objetivo de este repositorio es **unificar todas las entregas** en un solo espacio organizado, demostrando:
1. La construcción de la base de datos *Staging*.
2. La ejecución de scripts de creación, carga y validación.
3. La verificación de la calidad de los datos.
4. La documentación y evidencias del proceso.
